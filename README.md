C'est un projet qui simule une chaîne de production de Noël automatisée, dans un cadre scolaire, on utilise ici différent design pattern.

---

## Design Patterns Implémentés

On retrouve 6 patterns :

* **Facade (`Santa`)** : Interface unique pour piloter tout le processus.
* **Factory (`GiftFactory`)** : Création des objets cadeaux selon le type.
* **Decorator (`WrapppingGift`)** : Ajout de caractéristiques, cic le ruban.
* **Observer (`Workshop`)** : Notification automatique aux lutins.
* **Strategy (`DeliveryMethod`)** : Sélection du moyen de transport.
* **Command (`SantaCommand`)** : Encapsulation des actions pour une exécution centralisée.

---

## Utilisation

L'utilisation est simple. Il suffit de cloner le projet en local, d'avoir python d'installer sur votre machine et d'éxecuter le fichier principal Santa.py pour y suivre les instructions.
